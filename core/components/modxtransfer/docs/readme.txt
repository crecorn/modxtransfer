ModxTransfer - Import/Export for MODX 3
=======================================

ModxTransfer provides an easy-to-use interface for exporting and importing 
MODX elements and resources.

Features
--------
- Export Categories, Templates, Chunks, TVs, Snippets, Plugins
- Export Resources with Template Variable values
- Import with preview mode before executing
- Filter by category or template
- Preserve hierarchy and relationships
- MODX 3.x compatible

Requirements
------------
- MODX Revolution 3.0+
- PHP 7.4+

Installation
------------
1. Install via MODX Package Manager, or
2. Download and copy files to your MODX installation:
   - core/components/modxtransfer/
   - assets/components/modxtransfer/
3. Create namespace and system settings (see README.md for details)
